My Script

cpt-fractal.py runs in your terminal. The terminal will print a series of questions for the user to answer. Based on the user selections and input, the script will use the Turtle module to draw a fractal shape. The user is able to manually input all shape parameters, or choose to thave the script randomly generate them.
